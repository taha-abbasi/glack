# Glack

A native macOS app for Google Chat with Slack-style UX.

## The why

Google Chat's web app makes basic actions painful:
- Channel switching is slow
- Cmd-K barely works
- Search is unreliable
- No native Mac feel (menus, hotkeys, system integration)

Glack wraps Google Chat so it looks and behaves like Slack on macOS: fast channel/DM switching, working Cmd-K, real search, native menus, global hotkeys.

## Status

Very early. Scaffolding. Sign in with Google planned. Tech stack TBD (leaning Tauri for speed-to-MVP; native Swift in scope if Tauri's web-view route can't deliver the right feel).
